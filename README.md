# My Hyprland Config

Personal Hyprland configuration with pywal integration, custom SwayNC theme, and Waybar.

## Setup
```bash
git clone git@github.com:Maty156/.config.git ~/.config
cd ~/.config
# Then run any setup scripts if needed
```

## Features
- Pywal dynamic theming
- Glassmorphism SwayNC
- Clean Waybar with pywal colors
- Custom keybindings and scripts
